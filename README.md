# Ride Easy – Car Rental Portal (PHP & MySQL)

This project is a complete **Car Rental Management System** built using **PHP**, **MySQL**, HTML, CSS, JavaScript, and AdminLTE.

## Features
### User
- Signup/Login
- Browse vehicles
- Book vehicle
- View booking history

### Admin
- Manage brands
- Manage vehicles
- Approve/Cancel bookings
- Manage users

## Project Structure
```
Car-Rental-Portal-Using-PHP-and-MySQL-V-3.0/
└── carrental/
    ├── admin/
    ├── includes/
    ├── css/
    ├── js/
    └── index.php
```

## How to Run
1. Move project to XAMPP `htdocs`
2. Import `carrental.sql`
3. Update DB config
4. Run:
   - User: http://localhost/carrental/
   - Admin: http://localhost/carrental/admin/

## Custom Features Added
- Availability tracking
- “Not Available” when count = 0
- Email after admin approval
